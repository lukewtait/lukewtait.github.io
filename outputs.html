Online workshops

I have organised two workshops aimed at introducing dynamical systems in neuroscience. The contents for both are available online at the links below.

  • An introduction to modelling brain dynamics. This workshop is aimed at early career researchers in the field of neuroimaging with no quantitative background. It was originally meant to be delivered at Cardiff University Brain Research Imaging Center in early 2020, but was reformatted as an online workshop due to the COVID-19 pandemic. Topics include introductory dynamical systems, neural mass modelling, bifurcations, dynamics on networks, and model optimization. The workshop consists of four tutorials, each of which consists of a seminar and a practical session. There is some coding involved in the practical sessions, using either Matlab (original workshop) or Javascript (online version).
  • Introduction to dynamical systems: Applications in biology. This workshop was a 3 hour session delivered to A-level Maths students at Exeter Maths School. Topics include introductory dynamical systems, fitting a neural model to patch clamp experiments, and population dynamics (including modelling a zombie apocalypse). No coding is involved, but mathematical background to AS-level standard is required.

Software and Tools

Here I list software outputs and tools that I have developed.

  • Brain simulator webpage. This webpage contains some toy examples of brain simulations at different spatial scales.
  • Evaluate inverse methods. This Github repository contains the codes to evaluate the quality of a source reconstruction for different inverse methods as presented in Tait et al. (2020).
  • MEG-optimized HCP-230 brain atlas. This Github repository contains the MEG-optimized reduction of the HCP brain atlas presented in Tait et al. (2020).
  • Human brain simulator. This Github repository contains a code I developed to simulate large-scale human brain networks.
  • Data visualization toolbox. This Github repository contains a toolbox I developed for visualizing distributions of data in Matlab.
