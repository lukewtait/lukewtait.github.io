The video below shows the second session of the workshop "An Introduction to Modelling Brain Dynamics". In this session, Marinho Lopes (Cardiff University) introduces concepts in dynamical systems such as dynamics on networks, excitability, and bifurcations. He applies these concepts to modelling the propensity of brain networks to generate seizures in epilepsy. The slides can be found at the workshop's Github page.

After watching this seminar, you may wish to practice these skills in a practical session. The original session was run with a Matlab practical, for which the files can also be found at the the workshop's Github page. A slightly modified version of the practical which can be run directly from the browser is coming soon.
